4.4查询操作

4.4.1 单表查询

4.4.1.1单表查询

  • 标准语句：select * from 表名称;

  • 查日期

    • select current; mysql不需要加from子句
    • elect sysdate from dual; Oracle需要加from子句，有一个dual的伪装表

  • 查询显示列

    • select 列名1,列名2 from 表面;

  • 查询去重复

    • select distinct 列名 from 表名;

  • 查询用表达式

    • select 列名*12 as 年收入 from 表名;

  • 对空值进行处理

    • select sal*12+ifnull(comm,0) from emp; mysql用ifnull函数select
    • select sal*12+NVL(comm,0) from emp;oracle用NVL函数

  • 排序：order by

    • select * from 表名 order by 列名 排序方法;

  • 只显示前三行

    • select * from 表名 limit 3 #mysql用limit
    • oracle数据库要用rownum虚拟列来实现

4.4.3 多表查询

  • 等值连接
    • select a.,b. from a,b where a.id=b.id;
    • 内连接：select a.,b. from a join b on a.id=b.id;
  • 自身表连接
    • 把这个表看成A，又堪称B，但要出现两次
  • 不等值连接
  • 左右连接
    • 左右连接的区别
    • 左右链接的实现
      • select a.,b. from a left join b on a.id=b.id;
      • select e.,d. from dept d,emp e where d.deptno=e.deptno(+);

4.4.4 子查询

  • 子查询在from子句中，看成一张表
  • 子查询在where子句，把它看成一个数值:WHERE字段=(子查询)
  • 子查询在WHERE子句中，把它看成一组值:WHERE 子段IN(子查询)
