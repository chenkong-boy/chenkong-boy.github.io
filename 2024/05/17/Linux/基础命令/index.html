1、文件目录

在linux中一切皆为文件

应用、 进程、设备都是文件

1.1 文件类型

普通文件

  • 存放实际数据的文件，如文本文件、图像文件和可执行文件。
  • 使用连字符（-）表示。

目录

  • 组织和分层文件和目录的特殊文件。
  • 使用斜杠（/）表示。

链接文件

  • 指向另一个文件的特殊文件，充当快捷方式。
  • 有两种类型的链接文件：
    • 硬链接：指向同一物理文件的多个文件名。
    • 软链接（符号链接）：指向另一个文件的路径。
  • 分别使用 @ 和 -> 表示。

块设备文件

  • 表示块设备，如硬盘和分区。
  • 使用大写字母 B 表示。

字符设备文件

  • 表示字符设备，如键盘和鼠标。
  • 使用大写字母 C 表示。

套接字文件

  • 用于进程间通信。
  • 使用大写字母 S 表示。

管道文件

  • 用于进程之间的数据传输。
  • 使用大写字母 P 表示。

FIFO 文件

  • 先进先出（FIFO）队列，用于进程之间的数据传输。
  • 使用大写字母 F 表示。

其他文件类型

  • 设备节点文件：表示特定设备或设备的一部分。
  • 命名管道文件：类似于管道文件，但具有名称，可以在不同进程之间共享。

2、基础命令

强大的tab，可以

2.1 文件、目录

ls: 列出目录中的文件和子目录。

pwd: 查看当前路径

cd: 更改当前工作目录。

mkdir: 创建目录。

rmdir: 删除空目录。

touch: 创建或更新文件。

rm: 删除文件或目录。

cp: 复制文件或目录。

mv: 移动或重命名文件或目录。

ln: 创建链接文件。

2.2 文件查看和编辑

cat: 显示文件内容。

more: 分页显示文件内容。

less: 交互式分页显示文件内容。

head: 显示文件开头几行。

tail: 显示文件结尾几行。

vi: 文本编辑器。

nano: 文本编辑器。

vim: 高级文本编辑器

2.3 文本处理

grep: 搜索文本文件中匹配的模式。

sed: 流编辑器，用于查找和替换文本。

awk: 模式扫描和处理语言。

2.4 进程管理

ps: 显示正在运行的进程。

top: 显示系统资源使用情况和进程信息。

kill: 终止进程。

nice: 调整进程的优先级。

2.5 网络命令

ping: 测试网络连接。

traceroute: 跟踪网络数据包的路径。

netstat: 显示网络连接和统计信息。

ifconfig: 显示和配置网络接口。

2.6 系统信息和管理

uname: 显示系统信息，如内核版本和操作系统名称。

hostname: 显示或设置主机名。

free: 显示内存和交换空间使用情况。

df: 显示文件系统使用情况。

du: 显示目录和文件的大小。

sudo: 以超级用户权限运行命令。

2.7 其他常用命令

history: 显示历史命令。

clear: 清除终端屏幕。

man: 显示命令手册页。

help: 显示命令帮助信息。
